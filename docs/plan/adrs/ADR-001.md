# ADR-001: CLI con Picoli y GraalVM

## Estado
Aceptado

## Contexto
Necesitamos un CLI que sea:
- Rápido de ejecutar
- Distribuible como binario nativo
- Compatible con múltiples plataformas

## Decisión
Usar **Picoli** (framework CLI para Java) con **GraalVM Native Image** para compilación AOT.

## Consecuencias
- Compilación nativa produce binarios < 30MB
- Tiempo de startup < 100ms
- Requiere Java 25+
