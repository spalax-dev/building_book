# ADR-002: Diagramas embebidos en markdown

## Estado
Aceptado

## Contexto
Los diagramas deben:
- Ser editables como texto (version control friendly)
- Renderizarse automáticamente
- Incrustarse en documentos markdown

## Decisión
Usar **PlantUML** con directivas `@startuml` que generen archivos en `diagrams/`. Incrustar con `!["alt"](ruta.svg)`.

## Consecuencias
- Diagramas editables en texto plano
- Requiere PlantUML para visualización
- SVG generado automáticamente
