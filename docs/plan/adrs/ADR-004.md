# ADR-004: Rendering de Markdown con Commonmark-java

## Estado
Aceptado

## Contexto
El servidor web necesita convertir documentos markdown a HTML para visualizarlos en el navegador.

## Decisión
Usar **commonmark-java** para parsing y rendering de markdown a HTML.

## Consecuencias
- Parsing rápido y extensible
- Soporte para extensiones (tables, task lists, etc.)
- Integración con PlantUML para diagramas inline
