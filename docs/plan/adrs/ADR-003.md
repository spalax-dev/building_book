# ADR-003: Integración nativa con opencode

## Estado
Aceptado

## Contexto
El framework debe poder ser usado por agentes IA de forma autonomous.

## Decisión
Crear archivos de configuración en `.opencode/`:
- `agents.md`: Define workflow SDD para el agente
- `rules.md`: Reglas de documentación

## Consecuencias
- opencode reconoce el proyecto como SDD
- Agente puede consultar specs automáticamente
- Requiere mantener archivos actualizados
